# Sistema de Tareas Colaborativas en Tiempo Real

## Descripción

Aplicación web desarrollada para la materia de Tecnologías Emergentes utilizando servicios de Computación en la Nube.

El sistema permite registrar, visualizar, actualizar y eliminar tareas de manera colaborativa utilizando Supabase como backend en la nube.
