# Sistema de Registro de Tareas Colaborativas

Aplicación web desarrollada para la materia Tecnologías Emergentes.

## Tecnologías Utilizadas

- HTML
- CSS
- JavaScript
- Supabase
- Git
- GitHub
- Vercel

## Funcionalidades

- Crear tareas
- Listar tareas
- Actualizar estado
- Eliminar tareas
- Sincronización en tiempo real
